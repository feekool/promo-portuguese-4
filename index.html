⚡ Как запустить выбранную сборку

  1. 1. GitHub аккаунт Создай бесплатный профиль на github.com
  2. 2. Выбери сборку
    Открой шаблон: github.com/feekool/feekool.github.io и выбери нужную конфигурацию
  3. 3. Копируй репозиторий
    Нажми "Use this template" → "Create a new repository". Назови, например, "meu-portugues"
  4. 4. Включи GitHub Pages
    Settings → Pages → ветка main → Save. Через 30-60 сек форум онлайн!
  5. 5. Настрой под себя
    Редактируй `config.json` или HTML-файлы. Добавляй новые темы, меняй цвета, приглашай учеников.
💡 Pro tip: Хочешь кастомную сборку под твою методику? Напиши в Issues — соберём индивидуальную версию.
